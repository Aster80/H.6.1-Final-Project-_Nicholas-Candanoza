# Project README

## What is this?

This project is a Concrete Pad and Chain-Link Fence Estimator application. It helps users calculate the materials and estimated cost needed for construction projects like concrete slabs and fencing.

## What does it do?

The application allows users to input dimensions and receive calculations for concrete volume, fence length, and total estimated cost. It includes features like cost breakdowns, user input handling, and file outputs such as CSV or text files.

## How far did you get?

I would say I got about 70–85% complete. The main calculations and structure are working, but some features like UI improvements, full error handling, and additional refinements are still incomplete.

## What AI tools did you use?

I used ChatGPT to generate Java and HTML code, assist with calculations, debug errors, and help design the structure of the application.
